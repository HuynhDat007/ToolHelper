## Cấu hình trong Auto
### Cấu hình Đường dẫn LDPlayer
- Vào [**Cấu hình**]
- Nhấn [**Chọn**] ở dòng "_Đường dẫn LDPlayer9_"
<img width="439" height="931" alt="image" src="https://github.com/user-attachments/assets/17881f16-b6b3-4361-adf9-24aeba55d5b0" />

- Chọn folder(thư mục) chứa **LDPlayer**
  * Nếu cài LDPlayer9 thì chọn "LDPlayer9" như: "D:\LDPlayer\LDPlayer9"
  * Nếu cài LDPlayer5 thì chọn "LDPlayer64" như: "D:\LDPlayer\LDPlayer64"
<img width="946" height="533" alt="image" src="https://github.com/user-attachments/assets/b715cc9e-a888-4116-86c7-bd51989efba1" />

### Cấu hình Đường dẫn XuanZhi
- Nhấn [**Chọn**] ở dòng "_Đường dẫn XuanZhi9_"
- Chọn folder(thư mục) chứa **XuanZhi**
  * Nếu cài LDPlayer9 thì chọn "XuanZhi9" như: "C:\Users\XXXX\Documents\XuanZhi9"
  * Nếu cài LDPlayer5 thì chọn "XuanZhi64" như: "C:\Users\XXXX\Documents\XuanZhi9"
### Sau khi cấu hình các bước ở trên xong thì tắt hẳn LDPlayer xong mở lại
### Các tab khác có thể clone từ tab đã cấu hình chuẩn ở trên
